## Quick Start

1. **Clone this repository**
   ```bash
   git clone https://github.com/sravan1023/Drive-Sandbox.git
   cd Drive-Sandbox
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. Open your browser at  
   **http://localhost:5173**
